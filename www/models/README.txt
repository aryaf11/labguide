LabGuide — نماذج ثلاثية الأبعاد للتدريب في المنزل
================================================

ضع ملفات GLB هنا بأسماء ثابتة (بدون مسافات)، لتُحمَّل تلقائياً بدلاً من الشكل المبسّط:

  oscilloscope.glb
  breadboard.glb
  multimeter.glb
  power-supply.glb
  function-generator.glb
  spectrum.glb

لا يُستخدم ضغط Draco في الكود الحالي — صدّر من Blender بدون Draco أو فعّل فك الضغط لاحقاً.

إذا لم يوجد الملف أو فشل التحميل، يُستخدم النموذج البرمجي الحالي كبديل.

مصدر الأصول
------------
الأصول الواقعية عالية التفصيل تُنتج بالتصميم أو المسح ثم التصدير إلى GLB؛ الكود يحمّل هذه الملفات تلقائياً مع احتياطي مرئي عند غيابها.

النسخ إلى Android
------------------
بعد إضافة أو تغيير أي ملف تحت www/ (بما فيها www/models/*.glb)، من جذر المشروع:

  npx cap sync android

أو المزامنة العامة:

  npm run sync

لفتح المشروع بعد المزامنة: npm run android (مزامنة android + فتح Android Studio).

التفاعل (مقابض قابلة لللمس في وضع التدريب التفاعلي)
----------------------------------------------------
في ملف GLB، سمِّ مساحات المقابض بحيث يظهر في الاسم أحد:
knob / dial / rotary / button / potentiometer
ليُعرَف تلقائياً كزر دوّار يمكن النقر عليه لتدويره (مع تعطيل لف الكاميرا لأجزاء من الثانية عند اللمس).
